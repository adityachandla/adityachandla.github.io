My Projects

Distributed Chess Engine

A chess engine that evaluates best moves using Min-Max Algorithm. The computation can be distributed over as many nodes as there are possible moves at any point in the game. All client nodes evaluate a subset of positions and each give their estimates for the next move. In the end, a server nodes evaluates all the possible options provided by clients and then selects the best move. This project is implemented in Java.

Exceptional Model mining

Exceptional model mining is a sub-field of data mining that tries to find sub-models within a dataset that exhibit special properties. This project finds exceptional models in a housing dataset. This project also explores various ways in which we can look at the exceptionality of a sub-set of data. This project is implemented using Golang

Convex polygon covering

This project was implemented as part of a course on Geometric Algorithms. This project solves the problem of finding convex polygons to cover a larger polygon with holes. This project first triangulates the entire polygon and then, based on some heuristics, it tries to merge the triangles so that the resulting polygon is convex and we use fewest possible polygons. This project was tested on inputs provided as part of an international competition. This project is implemented using Java.
