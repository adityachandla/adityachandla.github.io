My Projects

Distributed Chess Engine

A chess engine that evaluates best moves using Min-Max Algorithm. The computation can be distributed over as many nodes as there are possible moves at any point in the game. All client nodes evaluate a subset of positions and each give their estimates for the next move. In the end, a server nodes evaluates all the possible options provided by clients and then selects the best move.
